EpicException
Base ancestor for all the infrastructural exceptions in the Epic's ecosystem.

[SerializableAttribute]
public class EpicException : InvalidOperationException
Remarks

Whenever an Epic's client will catch this exception (or any derived one) it will know that the problem occurred was related to the infrastructure.

EpicException extends InvalidOperationException becouse when the infrastructure has a problem, it's always due to the combined state of the infrastructural code itself (where infrastructural is everything but the domain model itself).
